# クラス変数について

クラス変数とは@@から始まる変数のこと大文字で始まるのはクラスを定義
するものでありクラス変数ではない

# EOF識別子について

EOFの識別子は<<識別子行から識別子行の直前までを文字列として扱う

```
`EOF`
```

これは終了識別子ではない
