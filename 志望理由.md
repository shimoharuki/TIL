### 企業の魅力ポイント

```
テクノロジーを駆使して、エンターテイメントやクライアントの要望に限りなく近いソリューションを提案している
たくさんの分野に触れ、新しい技術、情報を日々キャッチアップできる
部長や課長が存在せず横社会
```
### なぜ企業の魅力に感じたのか
```
子供の頃からエンターテイメントが好きで妥協せず全力で何かを全力で作り上げる姿勢にとても感動したから。
知らない分野にふれ積極的にキャッチアップしていくことで自分のできることが増えていき、アイデアを形にする手段が増えていく
たくさんの人とコミュニケーションが取れる事から、モチベーションを上げ続けたまま業務に取り組めそう
前職が年齢問わず全員で一丸となって働く会社だったから

```


### 紐付けられる原体験
```
・前職のUSJでエンターテイメントを作り上げるには一人では到底不可能で沢山の人が
協力し、同じ目的(USJに来園された方の体験価値を向上させる)に向かって全力で取り組んでいた。
・沢山のクルーがいたのでコミュケーションをとることでモチベーションを下げずにただ働くだけでなく
楽しんで働くことができた。
・向上心はあったが体を動かすハードワークだった事から将来現在と同じ様に働けなくなるのではという危機感があった

```
### 自分は何ができるか

```
・コミュニケーション能力を生かして業務を円滑に進め、互いにモチベーションを高め合いながら
同じ目標に向かって働ける
・成長意欲を生かして、沢山の知識を活かしてアプリをより良いものにし、
会社に貢献することができる。
```

### 志望理由
```
最新テクノロジーを駆使して、さまざまな顧客のニーズを満たしている点、
沢山の情報をキャッチアップできコミュニケーションを取りながら日々成長できる点に魅力を感じ貴社を志望しました。
```

```
前職ではUSJでアルバイトとして勤務をしており沢山のクルーと力を合わせてUSJに来園された方の
体験価値を向上するために日々業務を行っておりました。様々な声かけの仕方で来園された方の表情が明るくなった瞬間などに
日々やりがいを感じていました。
しかし室外に出て毎日体を動かすハードワークという事から
年齢を重ねるにつれて沢山のことをインプットしたとしてもアウトプットができなくなってしまうと危機感を感じました。
```
```
貴社の実績や社風を拝見させていただいた際に沢山の社員やクライアントとコミュニケーションをとり
全員が一丸となって全力で何かを作り上げる姿勢が私が前職で感じたやりがいと一致しています。
また沢山の知らない分野にふれ積極的にキャッチアップしていくことで自分のできることが増えていき、アイデアを形にする手段が増えていくことで
自分の成長が実感でき、学びながら業務を行えるのではないかと考えました。
```
